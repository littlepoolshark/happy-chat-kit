## [1.0.0] - 2025-03-17

### Added
- Initial stable release of HappyChatKit
- Core module: extensible SSE chat framework
- Preset module: ready-made UI with Ant Design X integration
- Support for custom protocols and renderers
- Markdown rendering with remark-directive support
- TypeScript strict mode support
- Complete documentation and examples

### Added
- First public release
- Core APIs: `ChatKit`, `HappyChatProvider`, `useHappyChatCore`
- Preset APIs: `useHappyChat`, `HappyChatMessageFlow`, `MarkdownRenderer`
- Dual entry points: `happy-chat-kit` and `happy-chat-kit/preset`
- ESM and CJS build outputs
- Basic documentation and examples
